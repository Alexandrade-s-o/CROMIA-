## üéûÔ∏è Proyect Overview
**CROMIA** is a premium digital portfolio built for creators who value minimalism and high-end interaction. This project showcases a curated selection of photography categories, featuring a unique "Slide to Unlock" mechanism inspired by classic iOS aesthetics to reveal color and detail.

## ‚ú® Key Features
- **Monochrome-to-Color Interaction**: A custom-built iPhone-style "Slide to Reveal" slider to unlock full-color high-resolution images.
- **Dynamic Portfolio Grid**: Responsive collage layouts optimized for various artistic categories (Advertising, Urbano, Moda, Editorial).
- **Premium Aesthetics**: Minimalist design with gold (`#D4A017`) and pastel pink (`#FFD1DC`) accents.
- **Mobile First**: Fully optimized touch interactions for mobile devices.
- **Digital Exhibition Modal**: immersive image viewing with smooth transitions and keyboard navigation.

## üöÄ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/)

### Local Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Alexandrade-s-o/CROMIA-.git
   cd CROMIA-
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
